# 인증서 릴레이 API

<Info>
  이 문서는 **준비 중**입니다.

  인증서 릴레이 API에 대한 자세한 내용은 [상담 문의](https://imprun.dev/contact)를 통해 확인하실 수 있습니다.
</Info>

## 개요

인증서 릴레이는 PC에 설치된 공동인증서를 모바일 앱으로 안전하게 전송하는 서비스입니다.

- **엔드포인트**: `POST /v2/certificates/upload/{certId}`
- **과금**: 무료
- **연동 방식**: REST API 또는 Android SDK
