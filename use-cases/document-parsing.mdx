## 개요

imprun의 문서 파싱 기능으로:

- 📄 부동산 등기부등본 정보 추출
- 🏢 건물 정보 자동 수집
- 📊 정확도 95% 이상의 인식률
- ⚡ 초 단위의 빠른 처리

스캔 이미지나 PDF에서 중요한 정보를 자동으로 추출합니다.

## 지원 문서

| 문서 타입 | 설명 | API 값 |
|----------|------|--------|
| **토지 등기부등본** | 소유자, 지번, 면적 등 | `land` |
| **건물 등기부등본** | 건물 주소, 면적, 소유자 | `building` |
| **집합건물** | 아파트 등 집합건물 정보 | `aggregate_building` |

## 사용 방법

### 1단계: 문서 이미지 준비

- 해상도: 최소 200 DPI
- 형식: JPG, PNG, PDF
- 크기: 최대 10MB

### 2단계: 파싱 요청

```bash
curl -X POST https://api.imprun.dev/v2/workflows \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "document_parsing",
    "params": {
      "documentType": "land",
      "imageUrl": "https://your-storage.com/document.jpg"
    }
  }'
```

### 3단계: 결과 수신

```json
{
  "id": "wf_parse123xyz",
  "status": "SUCCEEDED",
  "result": {
    "documentType": "land",
    "landNumber": "서울시 강남구 123-456",
    "owner": "홍길동",
    "area": "150.5",
    "usage": "주거",
    "confidence": 0.96
  }
}
```

## 예제: 토지 등기부등본 파싱

```bash
curl -X POST https://api.imprun.dev/v2/workflows \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "document_parsing",
    "params": {
      "documentType": "land",
      "imageUrl": "https://example.com/land-registry.jpg"
    }
  }'
```

**응답:**
```json
{
  "result": {
    "landNumber": "서울 강남구 신사동 123-456",
    "owner": "홍길동",
    "area": "250.5",
    "usage": "주거",
    "acquisitionDate": "2020-03-15",
    "confidence": 0.97
  }
}
```

## 비용

- **등기부등본 파싱**: ₩1,000 (건당)
- **높은 정확도 요청**: ₩500 추가

## 오류 처리

**문서 인식 실패:**
```json
{
  "status": "FAILED",
  "error": "Document not recognized",
  "advice": "사진 품질을 확인하고 다시 시도하세요"
}
```

**해결책:**
1. 문서를 정면에서 촬영
2. 그림자나 반사 제거
3. 해상도 200 DPI 이상 확보

## 사용 시나리오

### 🏠 부동산 거래 자동화
- 등기부등본 자동 수집
- 건물 정보 DB 구축
- 거래 절차 단축

### 🔍 법적 실사 자동화
- 부동산 소유권 검증
- 대량 문서 일괄 처리
- 감시 리포트 자동 생성

### 💼 재무 분석 자동화
- 자산 정보 자동 추출
- 가치 평가 시간 단축
- 데이터 오류 감소
